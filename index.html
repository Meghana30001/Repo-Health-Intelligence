Health Score Formula

Explainable AI — no black boxes
Health Score =
0.30×Complexity Drift—
0.25×Test Coverage—
0.25×Hotspot Risk—

= —
∿
Complexity Drift
Measures cyclomatic complexity, function length, and nesting depth changes commit-by-commit. Consistent increases signal tech debt accumulation.
radon · lizard · custom AST
✓
Test Coverage
Extracts coverage reports from CI artifacts and test folders. Higher coverage correlates with lower defect rates in production.
coverage.py · pytest · Jest
⚡
Hotspot Risk
Adam Tornhill's methodology: Hotspot = Code Churn × Complexity. Frequently edited AND complex files are statistically bug-prone.
Git log analysis · AST metrics
∞
Dependency Rot
Scans package.json, requirements.txt, and pom.xml for outdated versions, deprecated libraries, and unused dependencies against live registries.
npm audit · pip-audit · deps.dev
